# Exam Score Predictor

A Machine Learning project that predicts student exam scores based on:

- Age
- Gender
- Course
- Study Hours
- Class Attendance
- Internet Access
- Sleep Hours
- Sleep Quality
- Study Method
- Facility Rating
- Exam Difficulty

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook

## Model

- Decision Tree Regressor
- Random Forest Regressor

## Dataset

Student performance dataset containing academic and lifestyle factors.
